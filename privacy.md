# Privacy Policy

AICodePR does not store or retain any user code. When a pull request triggers a review, the diff is sent to the configured AI provider (DeepSeek or Anthropic) for analysis only. No code is logged, stored, or used for training purposes after the review completes.

GitHub App metadata (installation ID, repository name) is used solely for authentication and review delivery. No personal data is collected or shared with third parties beyond the AI provider required for code analysis.
